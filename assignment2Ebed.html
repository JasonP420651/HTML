Embedded

Embedded is where the CSS is put within the header or other section of the code where it is seperate to the code for the visual side of the webpage and is callable in any section of the code page

Embbedded is useful for reapeated instances of code being used, such as having every other paragraph green and the other reapeated paragraphs as blue. This needs to be reapeated within this one page as different pages have different colour arangments. In this case Embedded allows for different colours for each page and can be copied to other pages but is not as good for many webpages repeating colours or variations of CSS. This takes more space at the top of the code and can get too long on one webpage as more stuff is added to the page, and repeated code from page to page will increase the folder size.
