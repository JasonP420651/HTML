Box Model
Box Model demonstraites how CSS uses diffeing code to decide how the boxes interact. These are shown above and how they interact with eachother. Each lay on eachother as aditions to eachother, with the margin being transparent or a push from the layer behind it. The border contains the rest of the box, it can be coloured, and have differing types to allow for transitions wanted within the website. Padding is the gap between the content and border, this is used to stop contenet from being dificult to read as it is too close to the border to read well or for an effect such as hovering over may cause the text to move to the right using the padding to push it in a direction The Content is what is displayed within the box and can be text, images, drop boxes and anything which would need to be presented to the user
